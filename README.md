## API Reference

### Get all quotes

Returns a list of all the quotes.

```sh
  GET /api/v1/quotes
```

### Get a quote by id

Returns a quote by id.

```sh
  GET /api/v1/quotes/${id}
```

### Get a random quote

Returns a random quote.

```sh
  GET /api/v1/quotes/random
```
